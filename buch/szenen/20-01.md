# Kap 20, Szene 1 — Maren: Das Boot wächst

Maren auf der Werft, morgens. Das halbfertige Boot — Klaus' Boot, ihr Boot jetzt, ob sie will oder nicht. Sie hat gestern drei Planken eingesetzt. Heute sind es mehr. Fünf neue Planken, frisch, das Holz hell, Sägemehl auf dem Boden das sie nicht verursacht hat. Maren steht davor, Hände in den Hosentaschen, kaut auf der Unterlippe. Das Boot wächst. Nicht viel. Nicht offensichtlich. Aber mehr Planken als gestern, und das Sägemehl ist frisch, und die Nägel sind gesetzt, präzise, wie Klaus es gemacht hätte. Maren legt die Hand auf eine neue Planke. Warm. Natürlich warm. Alles in dieser Stadt. Sie drückt das Ohr auf das Holz. Unter der Planke, unter dem Boden, unter der Werft: ein Herzschlag. Langsam, regelmäßig, tief. Wie das Pochen im Garten. Wie der Puls den Alphina gespürt hat. Maren richtet sich auf. Denkt an Klaus. An seinen Umschlag. *Dreißig Tage warten.* Tag elf. Sie wartet. Das Boot wächst. Klaus ist tot und sein Boot wächst.
